## Description

Tomato Market Frontend

## Version

- React 18.2.0
- NodeJS 18.16.0

## Installation

```bash
$ npm ci
```

## Running the app

```bash
# development run
$ npm run dev
```

## Build

```bash
$ npm run build
```

## Lint / Prettier

```bash
# prettier
$ npm run prettier:check
$ npm run prettier:fix

# lint
$ npm run lint:check
$ npm run lint:fix
```

## Test

```bash
# test
$ npm run test
```
